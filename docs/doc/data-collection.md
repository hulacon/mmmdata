# Data Collection

This is an example subpage under the "About" parent page.

## Overview

The Multi-Modal Memory Dataset involves multiple phases of data collection:

1. **Screening and consent** – participants complete eligibility surveys
2. **Behavioral sessions** – memory tasks in the lab
3. **fMRI scanning** – brain imaging during memory encoding and retrieval
4. **Follow-up assessments** – optional longitudinal components

## Scanning Protocol

- **Scanner**: 3T Siemens Prisma
- **Sequences**: T1-weighted structural, T2-weighted functional (EPI)
- **Session duration**: ~90 minutes per participant

## Task Design

Participants encode and recall:
- Visual stimuli (images, scenes)
- Auditory stimuli (sounds, speech)
- Multimodal combinations

Tasks include both discrete trial-based experiments and naturalistic video viewing/recall.
