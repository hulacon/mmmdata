# DCM2BIDS Config

Modules in the `dcm2bids_config` package.

| Module | Description |
|--------|-------------|
| [cli](dcm2bids_config_cli) | CLI entry point for dcm2bids config generation. |
| [config_builder](dcm2bids_config_config_builder) | Build dcm2bids configuration dicts from structured session definitions. |
| [dicom_inspect](dcm2bids_config_dicom_inspect) | Inspect DICOM directories to extract fieldmap series information. |
| [overrides](dcm2bids_config_overrides) | Override file parsing for per-subject session idiosyncrasies. |
| [session_defs](dcm2bids_config_session_defs) | Session and task definitions for dcm2bids config generation. |
