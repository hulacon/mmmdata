# Analysis Methods

This is another example subpage under "Dataset Description."

## Preprocessing Pipeline

We use a combination of tools:

- **Anatomical preprocessing**: FreeSurfer, FSL
- **Functional preprocessing**: fMRIPrep
- **Quality control**: MRIQC, custom scripts

## Statistical Analysis

- **Univariate GLM**: FSL FEAT, SPM
- **Multivariate pattern analysis (MVPA)**: PyMVPA, Nilearn
- **Representational similarity analysis (RSA)**: custom Python/MATLAB code

## Reproducibility

All analysis code is version-controlled and documented. See the main repository for preprocessing and analysis scripts.
